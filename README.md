# 🏭 Ferrum IQ: Enterprise Control Room
**A Micro-Digital Twin & AI Optimizer for Mineral Processing**

## 🌍 Executive Summary
Ferrum IQ is an enterprise-grade Software as a Service (SaaS) prototype designed to optimize magnetic separation circuits in the mining and metallurgical industries. By utilizing Machine Learning (Random Forest Regressor), this digital twin helps plant managers maximize iron recovery while minimizing energy consumption and carbon tax liabilities.

## ✨ Enterprise Features
* **🎛️ AI Process Optimization:** Replaces manual guesswork with data-driven Field Strength (Gauss) and Drum Speed (RPM) recommendations.
* **🔴 Live SCADA Simulation:** Demonstrates real-time IoT data streaming from simulated plant sensors.
* **⚠️ Predictive Maintenance:** Automated warnings trigger when machines are pushed into high-risk degradation zones (e.g., motor overheating risk).
* **🏭 Fleet Dashboard:** A "God's Eye View" of multiple processing units, tracking the health and output of the entire plant line.
* **📈 Global Localization & Market Data:** Integrates with the Yahoo Finance API (`yfinance`) to pull live global iron ore stock prices, with dynamic currency conversion for international markets.

## 🛠️ Technology Stack
* **Frontend/UI:** Streamlit
* **Machine Learning:** Scikit-Learn (Random Forest)
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Plotly
* **Financial Data:** yFinance API
